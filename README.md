# 🚗 Agentic AI Smart Parking System

## 📌 Overview
This project implements an Agentic AI system that monitors, predicts, and optimizes parking using multiple intelligent agents.

## 🧠 Features
- Multi-agent architecture
- Real-time parking simulation
- Demand prediction
- Congestion analysis
- Decision optimization
- Streamlit dashboard

## ⚙️ Tech Stack
- Python
- LangChain
- Google Gemini API
- Streamlit
- Pandas

## 🚀 How to Run

```bash
pip install -r requirement.txt
streamlit run ui/adk_dashboard.py
